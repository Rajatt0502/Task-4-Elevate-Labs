# Task-4-Elevate-Labs

🌟 Mobile-Friendly Website with CSS Media Queries 🌟

Make your website look amazing on any device—from huge desktops to tiny phones! This project transforms a static, desktop-only page into a fully responsive, mobile-friendly layout using CSS media queries.


🚀 Features

Responsive Layout: Columns stack vertically on smaller screens

Flexible Navigation: Menus collapse or rearrange for mobile

Smart Images: Automatically scale to fit any container

Readable Fonts: Text adjusts for smaller screens

Smooth Experience: No overflow or awkward scrolling



🛠️ Technologies Used

HTML5 ✨

CSS3 🎨 (with media queries)



📦 Installation

Clone the repo:

git clone https://github.com/your-username/mobile-friendly-website.git

Open index.html in your favorite browser.

Modify styles in CSS as needed.



🌐 Usage

Resize your browser or open DevTools to simulate mobile devices

Watch the layout transform magically for smaller screens
